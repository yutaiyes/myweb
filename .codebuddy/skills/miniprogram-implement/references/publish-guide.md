# 小程序发布指引

## 发布流程概览

```
H5 预览开发 → 配置 AppID/密钥 → 生成体验版二维码 → 提交审核 → 正式发布
```

## 阶段一：配置体验版小程序

### 触发时机

当用户表达以下意图时，展示配置引导：
- "想发布小程序"
- "生成体验版"
- "上线小程序"
- "怎么让别人用"

### Agent 响应模板

```
要生成体验版小程序，需要先配置 AppID 和代码上传密钥。

请点击上方「发布」按钮，在发布面板中填写配置信息。
```

---

## 阶段二：获取 AppID 和密钥

### 前置条件

- 已注册微信小程序账号（非个人订阅号）
- 账号已完成认证（企业/个体工商户/个人）

### 获取 AppID

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入「开发管理」→「开发设置」
3. 在「开发者ID」栏目找到 **AppID(小程序ID)**

### 获取代码上传密钥

1. 在「开发设置」页面，找到「小程序代码上传」栏目
2. **关闭 IP 白名单**：点击「IP白名单」右侧的「关闭」按钮（必须关闭，否则云端上传会失败）
3. 点击「生成」按钮（首次）或「重置」按钮
4. 下载私钥文件（.key 文件）
5. **重要**：私钥文件仅生成时可下载一次，请妥善保存

**密钥格式说明**：

下载的 .key 文件用文本编辑器打开后，内容格式如下：

```
-----BEGIN RSA PRIVATE KEY-----
MIIEpAIBAAKCAQEA...（中间是一长串字符）...
-----END RSA PRIVATE KEY-----
```

填写时需要**完整复制**，包括开头的 `-----BEGIN RSA PRIVATE KEY-----` 和结尾的 `-----END RSA PRIVATE KEY-----`。

### Agent 响应模板（用户询问如何获取时）

```
获取 AppID 和密钥的步骤：

1. 登录微信公众平台 mp.weixin.qq.com
2. 进入「开发管理」→「开发设置」
3. AppID 在「开发者ID」栏目
4. 代码上传密钥在「小程序代码上传」栏目：
   - 先关闭「IP白名单」（必须关闭，否则云端上传会失败）
   - 点击「生成」下载私钥文件（.key）

密钥格式说明：用文本编辑器打开 .key 文件，完整复制内容（包括 -----BEGIN RSA PRIVATE KEY----- 和 -----END RSA PRIVATE KEY-----）。

请按照官网流程操作，获取后在发布面板中填写。
```

### Agent 响应模板（用户重复询问/发截图问下一步时）

```
请查看上文指引，以微信公众平台官网流程为准。

如果已获取到 AppID 和密钥，请在发布面板中填写配置信息。
```

## 阶段三：生成体验版二维码

### 触发条件

- AppID 和密钥已配置
- 代码已完成开发

### 后端 API

```
POST /wx-miniprogram/preview
```

### 成功响应

展示体验版二维码，并提示：

```
体验版小程序已生成！

扫描二维码即可预览。如需添加体验成员，请在微信公众平台「成员管理」中添加。
```

---

## 阶段四：提交审核与发布

### 提交审核前置条件

1. **服务类目**：在「设置」→「基本设置」→「服务类目」中选择
2. **隐私协议**：在「设置」→「基本设置」→「用户隐私保护指引」中填写
3. **域名配置**：在「开发设置」→「服务器域名」中配置 request 合法域名

### 提交审核步骤

1. 登录微信公众平台
2. 进入「版本管理」
3. 在「开发版本」中点击「提交审核」
4. 填写版本说明，确认提交

### Agent 响应模板

```
提交审核前，请确认以下配置已完成：

1. 服务类目：「设置」→「基本设置」→「服务类目」
2. 隐私协议：「设置」→「基本设置」→「用户隐私保护指引」
3. 服务器域名：「开发设置」→「服务器域名」

完成后在「版本管理」→「开发版本」中点击「提交审核」。

审核通常需要 1-7 个工作日，请耐心等待。
```

## 常见问题

### Q: 没有小程序账号怎么办？

```
请先在微信公众平台 mp.weixin.qq.com 注册小程序账号。

注册类型说明：
- 个人：功能受限，无法使用支付等能力
- 企业/个体工商户：功能完整，需提交资质认证

注册完成后再进行发布配置。
```

### Q: 密钥文件丢失怎么办？

```
在微信公众平台「开发设置」→「小程序代码上传」中点击「重置」，重新生成密钥。

注意：重置后旧密钥将失效。
```

### Q: 审核被拒怎么办？

```
请查看微信公众平台的审核反馈，常见原因：

1. 服务类目与内容不符
2. 缺少必要的资质证明
3. 隐私协议不完整
4. 存在违规内容

根据反馈修改后重新提交审核。
```
