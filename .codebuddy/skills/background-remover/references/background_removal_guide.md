# AI 去底工具使用指南（`rembg`）

## 工具概述

本工具使用 `rembg` 的 **AI 前景分割**能力自动识别主体与背景，并输出带透明通道（alpha）的 PNG。

适用场景：
- 像素画、精灵图、道具图标、UI 元素去底
- 需要快速把素材变成可直接用于游戏引擎的透明贴图

不适用/不再支持的旧参数：
- 魔棒算法（Flood Fill）相关的 `tolerance` / `seeds` / `preview` / `batch` 参数

## 基本用法

脚本位置：`scripts/remove_background.py`

## 处理流程（当前实现）

该脚本的实际处理流程是：

1. **加白边**：用 PIL 打开输入图片，给图片四周添加 100px 白边（用于增强主体边缘识别）
2. **AI 分割**：调用 `rembg`（`isnet-general-use` 模型）进行前景分割
3. **边界伪影去除**（核心优化）：
   - 裁剪回原图大小，丢弃白边区域内的任何伪影
   - 提取 Alpha 通道，进行二值化处理
   - 查找最大轮廓并填充（防止内部花纹丢失）
   - 边缘收缩（2 次腐蚀）去除细微白边
   - 高斯模糊平滑锯齿
4. **合成输出**：将处理后的蒙版与原图 RGB 合并
5. **裁剪透明区域**：执行 `getbbox()` 裁剪，去掉多余透明像素
6. **保存输出 PNG**

### 1) 单文件处理

```bash
python3 scripts/remove_background.py input.png
```

### 2) 指定输出路径

```bash
python3 scripts/remove_background.py input.png -o output.png
```

### 3) 批量处理（推荐用 shell 循环）

该脚本每次只处理一个输入文件；批量处理请交给 shell：

```bash
# 处理当前目录全部 png
for f in *.png; do
  python3 scripts/remove_background.py "$f" -o "${f%.*}_ai.png"
done
```

你也可以在子目录递归处理（谨慎执行，建议先在小目录测试）：

```bash
# macOS/Linux: 递归查找并处理 png
find . -name "*.png" -maxdepth 2 -print0 | while IFS= read -r -d '' f; do
  python3 scripts/remove_background.py "$f" -o "${f%.*}_ai.png"
done
```

## 输出规则

- 输出为 **PNG（带透明通道）**。
- **强烈建议总是使用 `-o/--output` 指定输出文件名**，并确保以 `.png` 结尾（方便工具链/引擎识别透明通道）。
- 若不指定 `-o`，脚本会用一个简单规则推导输出名：`args.input.replace(".", "_clean.")`。
- 原文件不会被覆盖（除非你手动把 `-o` 指向同名路径）。

## 常见问题（Troubleshooting）

### 1) 边缘有白边/黑边（Halo）

**原因**：源图抗锯齿或背景与主体颜色过于接近，分割边界不稳定。

**建议**：
- 尽量使用背景更干净、对比更强的源图
- 使用更高分辨率素材再去底（缩小到游戏尺寸时边缘会更干净）
- 必要时用图像编辑器做一轮手工修边（1-2 分钟通常足够）

### 2) 主体被削掉/缺失

**原因**：模型把主体某些部分误判为背景。

**建议**：
- 换一张背景更简单、主体更清晰的源图
- 避免主体与背景同色系
- 如果是 UI 元素，尽量生成“纯色背景 + 高对比主体”的版本再处理

### 3) 背景没完全去掉

**原因**：图像背景复杂（纹理/渐变/噪点），模型分割结果保留了部分背景。

**建议**：
- 优先选择背景更纯净的源图
- 先做简单预处理（模糊背景、提亮主体、增强对比）再去底

## 最佳实践

- **源图优先**：背景越干净、主体越清晰，去底效果越稳定。
- **批处理先抽样**：先挑 1-2 张代表图测试效果，再批量跑全量素材。
- **输出命名**：建议统一 `*_clean.png`，便于追踪是否经过去底处理。

## 技术细节

### 边界伪影去除原理

旧版本直接对带白边的图像进行处理，可能在边缘区域产生伪影。新版本的核心改进：

1. **裁剪优先**：AI 分割后，先把结果裁剪回原图大小，直接丢弃白边区域的所有像素
2. **轮廓填充**：找到最大轮廓并填充，防止主体内部花纹被误删
3. **边缘收缩**：通过腐蚀操作去除残留的细微白边
4. **平滑处理**：高斯模糊消除锯齿，使边缘更自然
